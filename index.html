🔗 Link einfügen

Verlinkt auf eine andere Seite deines Projekts. Im Browser müssen alle HTML-Dateien im gleichen Ordner liegen.
Springt zu einem Anker (⚓) auf der aktuellen Seite.

⚓ Anker setzen

Ein Anker ist eine Sprungmarke. Links können direkt zu ihr springen. Im Editor erscheint ein orangenes Badge.

🖼 Bild einfügen

Empfehlung: Lade das Bild zuerst herunter und füge es dann über „💾 Datei vom Computer" ein. Das funktioniert immer zuverlässig – egal von welcher Webseite das Bild stammt.
So geht es:
1. Gehe auf eine Bildseite (z. B. Pixabay, Unsplash oder Wikipedia)
2. Lade das Bild herunter (Rechtsklick → „Bild speichern unter…")
3. Wähle die gespeicherte Datei hier aus
✅ Bereit – klicke auf „Einfügen"
So geht es:
1. Klicke im Browser mit der rechten Maustaste auf ein Bild → „Bild kopieren"
2. Klicke in das Feld unten und drücke Strg+V
Tipp: Klappt am besten bei Wikipedia-Bildern. Bei Pixabay/Unsplash lieber herunterladen.
📋 Hier klicken, dann Strg+V drücken
✅ Bild bereit – klicke auf „Einfügen"
⚠️ Achtung: Viele Webseiten (z. B. Pixabay, Unsplash) blockieren das direkte Einbinden über URLs. Am zuverlässigsten funktioniert das nur bei Wikipedia-Bildern (commons.wikimedia.org).
Falls die Vorschau rot erscheint → bitte Bild herunterladen und über „Datei vom Computer" einfügen.
